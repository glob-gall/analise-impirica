QUAL O DOBRO?
Escreva um programa que leia, como entrada, o valor de um número inteiro positivo.

Como saída, imprima o dobro do valor digitado.

Dicas
Na linha 4, use o comando num = int(input("Digite o numero: ")) para ler um valor digitado pelo usuário via teclado. Ele funciona assim:
O programa imprime o texto "Digite o numero: " na tela do computador, a fim de orientar o usuário.
O usuário digita um número e pressiona a tecla ENTER.
O valor digitado pelo usuário é lido pelo comando input(), mas ele é entendido como texto pelo programa.
O texto é convertido em número inteiro pelo comando int().
O número inteiro é armazenado na variável num.
Exemplos de Entrada e Saída
Entrada	
11
Saída	
22
Entrada	
256
Saída	
512
Entrada	
5678
Saída	
11356
